# Conversor de Moedas

Aplicação web responsiva para conversão de moedas em tempo real, desenvolvida com HTML5, Bootstrap 5 e JavaScript (ES6+), consumindo uma API pública de câmbio para obtenção de taxas atualizadas automaticamente.

---

## Visão Geral

Este projeto permite converter valores entre diferentes moedas de forma rápida e precisa, utilizando taxas de câmbio em tempo real.

A aplicação utiliza o dólar americano (USD) como base para garantir consistência nas conversões.

---

## Demonstração

Interface simples, responsiva e otimizada para dispositivos móveis, com foco em usabilidade e experiência do usuário.

---

## Funcionalidades

- Conversão de moedas em tempo real  
- Integração com API pública de câmbio  
- Atualização automática das taxas  
- Cálculo baseado em moeda base (USD)  
- Layout responsivo (mobile-first)  
- Interface leve e otimizada  
- Exibição da taxa utilizada na conversão  
- Exibição da data da última atualização  

---

## Tecnologias Utilizadas

- HTML5  
- CSS3 (Bootstrap 5)  
- JavaScript (ES6+)  
- API: https://open.er-api.com
